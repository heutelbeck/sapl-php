# sapl-php

SAPL Policy Enforcement Point (PEP) for PHP and Symfony.

Enforces [SAPL](https://sapl.io) policies in a Symfony application against a SAPL
Policy Decision Point (PDP), covering single decisions and streaming
subscriptions.

## Requirements

- PHP 8.3+
- Symfony 7.3+

## Development

```
composer install
composer test
composer stan
composer cs-check
```
